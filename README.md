# Dovi Jobs — Curated Job Dashboard for Madeline M. Dovi

A daily-refreshed dashboard of communications, public relations, writing, copywriting, journalism, editorial, and professional marketing roles tailored for **Madeline M. Dovi**.

🔗 **Live site:** [bgraham14072.github.io/dovi-jobs](https://bgraham14072.github.io/dovi-jobs/)

## What's inside

- **Curated jobs** across three regions: Buffalo / WNY (in-person + hybrid), NYC metro, and Remote
- **Match scoring** (0-100) against Maddie's resume and cover-letter materials
- **Tailored cover-letter openers** for each role, copy-to-clipboard ready
- **Application tracker** with status (saved → applied → interview → offer) stored locally in your browser
- **Filters**: region, work mode, minimum salary, minimum match score, application status
- **Writing portfolio sidebar** linking to LinkedIn, Substack, The Riff, and UB Now
- **Search** across title, company, skills, and description
- **Full-time roles with healthcare benefits only**

## Data refresh

The jobs feed (`jobs.json`) is refreshed daily at **7:00 AM Eastern** via a scheduled Perplexity Computer task that re-runs the curation pipeline against LinkedIn, Indeed, HERC, and partner job boards.
